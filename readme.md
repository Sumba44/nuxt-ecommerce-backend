# Nuxt Ecommerce Backend

Node.js e-commerce backend system for my own project (NUXT.js e-commerce) https://github.com/Sumba44/nuxt-ecommerce-fe

Express.js  
Sequelize
MySQL

### Installation

```
npm install
```

Development

```
npm run dev
```
